## Постановка задачи.
### Даны α - регулярное выражение, задающее язык L, буква x и натуральное число k. Вывести длину кратчайшего слова из языка L, содержащего префикс x^k.

### Решение.
#### Есть 2 массива, с помощью которых всё пересчитывается: 
* `can_do_xk[i]` - лежит ли слово x^i в языке L.
* `min_len_prefix[i]` - минимальное слово, лежащее в языке L, с префиксом x^i.

#### Идем по регулярному выражению и поддерживаем стек, элементы которого: `object_stack`.
* Когда появляется '+', нужно взять минимумы для `min_len_prefix` и OR для `can_do_xk[i]`
* При умножении немного сложнее. Всё зависит от левого слова: какой у него префикс достижим.
* Во время звезды Клини, получается аналог задачи о рюкзаке с повторениями: нужно проверить, можно ли получить слово с префиксом x^i.

#### Для работы нужно заполнить поля `info_class` (через поток ввода или через файл) и запустить `calculate(info_class)`. Результатом будет целое число или `inf`.
